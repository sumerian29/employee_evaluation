Detailed Description

Employee Evaluation System (EES)
This program is an intelligent, fully integrated system developed for Thi Qar Oil Company to accurately and transparently assess employee performance within the framework of ISO 9001 and institutional excellence standards.

The system provides the following key features:

Manual candidate entry with instant list updates.

Accurate evaluation scoring (1–10 scale) through an intuitive and elegant graphical interface.

Automatic distribution of scores across four main performance categories:

Professional conduct and discipline (25 points)

Technical competence and quality of work (35 points)

Initiative and development (20 points)

Corporate values and teamwork (20 points)

Automatic total score calculation (out of 100) using precise mathematical logic without fractions.

Automatic saving of all evaluations into structured JSON files for easy tracking and retrieval.

Professional PDF report generation, including the company logo, ideal employee identification, and detailed section summaries—all fully formatted in Arabic.

Automatic background music playback and display of the company logo at startup to enhance corporate identity.

Smooth and responsive user interface with a light blue gradient theme, designed to run seamlessly on any Windows system as a standalone executable (no Python required).
